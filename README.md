# Installation
Requirement
- Jupyter/IPython Notebook

```console
$ git clone https://github.com/sn0wle0pard/tracer.git
$ cd tracer
$ pip install -e .
$ jupyter nbextension install --py --symlink --sys-prefix tracer
$ jupyter nbextension enable --py --sys-prefix tracer
```
